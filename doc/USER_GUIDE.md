# Guide Utilisateur Plume

Bienvenue dans le manuel complet de Plume.

## 1. Démarrage Rapide

### Créer un Nouveau Projet
Au lancement, Plume charge le dernier projet ouvert ou un projet par défaut.
*   Allez dans **Menu > Nouveau Projet**.
*   Entrez le titre, le genre et un résumé.
*   Le projet est automatiquement sauvegardé dans votre navigateur (LocalStorage).

### Sauvegarde et Export
Plume sauvegarde automatiquement vos modifications. Cependant, il est CRUCIAL d'exporter régulièrement votre travail.
*   **Export JSON** : Sauvegarde complète incluant tout (texte, notes, enquête). À utiliser pour les backups.
*   **Export DOCX/PDF** : Pour imprimer ou partager votre manuscrit.

## 2. L'Interface d'Écriture

### Structure Narrative
Plume organise votre histoire en **Actes > Chapitres > Scènes**.
*   **Ajouter un Acte** : Cliquez sur le "+" en bas de la colonne de gauche.
*   **Déplacer** : Utilisez le Drag & Drop pour réorganiser vos scènes et chapitres.

### Modes d'Affichage
*   **Mode Éditeur** : L'interface standard avec la barre latérale.
*   **Focus Mode** : Masque tout sauf le texte. Idéal pour la rédaction pure.
    *   *Raccourci* : Cliquez sur l'icône "Œil" ou "Focus" dans la barre d'outils.
*   **Split View** : Divise l'écran en deux pour afficher deux documents côte à côte (ex: Chapitre 1 à gauche, Fiche personnage à droite).

## 3. Outils d'Analyse (Menu "Stats")

Plume analyse votre texte en temps réel pour vous aider à améliorer votre style.

### Analyse de Lisibilité (Flesch Score)
Un score de 0 à 100 vous indique si votre texte est facile à lire.
*   **90-100** : Très facile (Enfant).
*   **60-70** : Standard.
*   **0-30** : Très difficile (Universitaire/Technique).

### Répétitions
L'outil détecte les mots répétés trop souvent dans une même scène.
*   Les répétitions de mots courants (le, la, de...) sont ignorées.
*   Cliquez sur un mot dans la liste pour surligner toutes ses occurrences dans le texte.

## 4. L'Investigation Board (Tableau d'Enquête)

C'est l'outil phare pour les romans policiers, thriller ou à mystère. Il permet de gérer des "Cas" (Cases) complexes.

### Concepts de Base
*   **Cas (Case)** : Une affaire majeure (ex: "Le meurtre du Comte").
*   **Fait (Fact)** : Tout élément d'information (Indice, Témoignage, Alibi).
*   **Personnage (Character)** : Vos suspects et témoins.

### Comment l'utiliser ?
1.  **Créer un Cas** : Dans l'onglet Investigation, créez une nouvelle affaire.
2.  **Ajouter des Faits** : Listez les indices trouvés (ex: "Couteau ensanglanté", "Lettre de menace").
3.  **Matrice MMO (Mobile, Moyens, Opportunité)** :
    *   Sélectionnez un Suspect et une Victime.
    *   Évaluez (de 0 à 10) si le suspect avait le **Mobile**, les **Moyens** et l'**Opportunité**.
    *   Liez des **Faits** pour justifier chaque note (ex: Lier "Lettre de menace" au "Mobile").
4.  **Matrice de Connaissance** :
    *   Définissez qui sait quoi. Est-ce que le Détective sait pour le couteau ? Est-ce que le Tueur sait que le Détective sait ?

## 5. La Timeline (Chronologie)

Plume propose deux vues pour gérer le temps.

### Timeline Classique
Une liste verticale des événements, utile pour vérifier la cohérence des dates.

### Timeline "Metro"
Une visualisation graphique où chaque personnage est une "ligne de métro" et chaque scène une "station".
*   Permet de voir instantanément quel personnage apparaît dans quelle scène.
*   Permet de visualiser les intrigues secondaires qui se croisent.
